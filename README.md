# Petris
======

A simple Java reinterpretation of the classic game "Tetris". No pictures, only 100% java code (at the moment).

## How to play

Right/Left : Move piece
Up/Down : Rotate piece
Space : Drop down piece
D : one line down
S : use power
Y : change power
T : change "theme"
P/Esc : pause/resume
N : start a new game
